# Aurevon

A premium music player experience.

## Features
- Spotify & YouTube integration
- AI-powered recommendations (Gemini & Groq)
- Mood-based playlists
- Seamless playback and queuing

## Setup
1. Clone the repository.
2. Install dependencies: `npm install` and `cd backend && npm install`.
3. Configure your API keys in `.env` (use `.env.example` as a template).
4. Run the development server: `npm run dev`.
5. Start the backend: `npm run server` (in the `backend` directory).
